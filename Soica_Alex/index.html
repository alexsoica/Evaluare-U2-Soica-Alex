Structura retelei personale si accesul la internet

Caracteristicile conexiunii la retea

  1. Adresa MAC:
    - Este folosita pentru recunoasterea dispozitivelor;
    - Are forma hexazecimala de tipul 00:1A:2B:3C:4D:5E.
  2. Adresa IP:
    - Este adresa logica a dispozitivului in retea sau pe internet;
    - Poate fi statica(setata manual) sau dinamica(setata automat).
  3. Debit la download:
    - Este viteza cu care datele sunt primite de dispozitiv;
    - Se masoara in Mbps.
  4. Debit la upload:
    - Este viteza cu care datele sunt trimise de dispozitiv;
    - Se masoara tot in Mbps.
